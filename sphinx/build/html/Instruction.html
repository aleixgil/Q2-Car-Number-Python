Mòdul Instruction¶

class instruction.Adc[source]¶

Suma registre-registre amb carry.

class instruction.Add[source]¶

Suma registre-registre sense carry.

class instruction.And[source]¶

And registre-registre.

class instruction.Brbc[source]¶

Salta a adreça de programa si cert bit de FLAGS és 0.

class instruction.Brbs[source]¶

Salta a adreça de programa si cert bit de FLAGS és 1.

class instruction.Break[source]¶

Atura la simulació i acaba l’execució del programa simulador.

exception instruction.BreakException[source]¶

S’utilitza per a aturar la simulació.

class instruction.Eor[source]¶

Or exclusiva registre-registre.

class instruction.In[source]¶

Quan s’aplica al port 0x0, llegeix un caràcter del teclat.

class instruction.InstRunner[source]¶

Superclasse abstracta de totes les classes lligades a instruccions del simulador.

execute(instr, state)[source]¶

Mètode abstracte redefinit a cada subclasse que executa l’ordre corresponent i modifica l’estat del MCU segons les especificacions de la instrucció.

match(instr)[source]¶

Mètode abstracte redefi nit a cada subclasse que retorna True si la classe de la instrucció pot executar l’ordre corresponent.

class instruction.Ldi[source]¶

Assigna valor a registre.

class instruction.Lds[source]¶

Còpia memòria a registre.

class instruction.Lsr[source]¶

Desplaçament dreta registre.

class instruction.Mov[source]¶

Còpia registre-registre.

class instruction.Nop[source]¶

No fa res. És la instrucció nul·la.

class instruction.Or[source]¶

Or registre-registre.

class instruction.Out[source]¶

S’usa per escriure per la pantalla. Quan el port és: 0x0: Escriu el valor del registre corresponent pel terminal en base 10. 0x1: Escriu el valor del registre corresponent pel terminal en base 16. 0x2: Escriu el valor del registre assumint que correspon a la codificació UTF d’un caràcter.

class instruction.Rjmp[source]¶

Salt relatiu a una nova instrucció.

class instruction.Sts[source]¶

Còpia registre a memòria.

class instruction.Sub[source]¶

Resta registre-registre sense carry.

class instruction.Subi[source]¶

Resta registre-constant sense carry.
